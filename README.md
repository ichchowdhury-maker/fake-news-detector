# Fake News Detector

A Machine Learning project that detects whether a news statement is Real or Fake.

## Features
- News classification
- NLP using TF-IDF
- Machine Learning model
- Streamlit web interface

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

1. Install requirements
pip install -r requirements.txt

2. Run the app
streamlit run app.py
